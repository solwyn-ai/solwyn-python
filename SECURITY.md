# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Solwyn SDK, please report it responsibly.

**Email:** dev@solwyn.io

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Scope

This policy covers the `solwyn` Python package (this repository). For vulnerabilities in the Solwyn Cloud API or Dashboard, contact the same email address.
